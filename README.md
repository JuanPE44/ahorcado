## Juego del Ahorcado

Este es un proyecto del juego del ahorcado desarrollado con HTML, CSS y JavaScript. Es una versión clásica del popular juego de adivinanzas en el que los jugadores intentan adivinar una palabra oculta antes de que se complete la representación gráfica de un ahorcado.

## Demostración

![ahorcado](https://github.com/JuanPE44/ahorcado/assets/89142353/29508bab-9768-47fa-9172-1e6ddfdf883c)

## Características

- Representación gráfica del ahorcado en cada turno incorrecto.
- Retroalimentación visual de las letras adivinadas en la palabra oculta.
- Se juega de a dos personas.

## Uso

1. Descarga o clona este repositorio en tu máquina local.
2. Abre el archivo `index.html` en tu navegador web preferido.
